Issue Tracker

Assign a Project Manager to a project, and assign Developers to the individual tickets. Available roles are Admin, ProjectManager, Developer, and Submitter. Rolodex Pro

Rolodex Pro is an ASP.NET (.NET 6) MVC multi-tenant contact management application that allows users to manage their contacts in categories. Whale Calculator

Whale Calculator is a loan calculator built with an ASP.NET Web API (.NET 6) back-end microservice and a Vue3 front-end user interface deployed to Microsoft Azure. Visitors enter the loan amount, term in months, and rate and get a breakdown of the monthly payment, total interest, and total loan cost. The payment schedule displays in a table with an estimate of how each payment is applied. View the live demo at https://whalecalc.jgnovak.dev.
