Go Command Line Tool:

Go command line application created to sync Active Directory group members with appropriate role in Intel vPro EMA Console. Run as a scheduled task from the EMA Console server, emasync ensures that members of Active Directory IT Security Groups are added to the correct EMA Security Roles in EMA Console by comparing Active Directory group membership with EMA Console Role membership via REST API. Missing accounts from designated Active Directory IT Security Groups are added via REST API to the corresponding EMA Security Role in EMA Console. Accounts that have been removed from an Active Directory IT Security Group are removed from the corresponding EMA Security Role via REST API.
